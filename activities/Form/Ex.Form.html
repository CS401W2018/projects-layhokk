What are different ways to style your form?

You can style forms by targeting inputs, buttons, and labels individually to improve readability. Using simple CSS properties like padding, margin, border-radius, and colors creates a clean look.

Why does the professor like to use flex?

Flexbox makes it easy to create responsive layouts without needing complex CSS or media queries. Flex items can automatically adjust their size and wrap based on the available space, which is perfect for forms that need to adapt to different screen sizes.

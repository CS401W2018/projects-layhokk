1. Why is the viewport meta tag important?

The viewport meta tag is crucial for responsive web design because it helps control how a webpage is displayed on different devices, particularly mobile devices

2. How can you adjust the CSS for printing?

To adjust CSS for printing, you can use media queries specifically for print. This allows you to define styles that will only be applied when a webpage is printed.

3. When would you want to adjust the CSS for printing?

I would adjust the CSS for printing to ensure that the printed version of a webpage is clear, concise, and optimized for paper, focusing on readability, content relevance, and resource conservation

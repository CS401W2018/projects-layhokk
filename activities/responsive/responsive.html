1. Why do we need a viewport meta tag?

It tells the browser to fit the page's width to the screen size and ensures the initial zoom level.

2. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

The priority is determined by the order so flex-direction-row is the priority.

3. How do you change the CSS for a web page displayed on a phone?

To change the CSS specifically for a webpage displayed on a phone, you can use CSS media queries. For a phone, you typically target smaller screen widths, such as 800px or less

1. What is the difference between block tags and inline tags.

Block tags create a block-level element which means they start on a new line and take up the full width of their parent container by default.

Inline tags create inline-level elememt which means they do not start on a new line and only take up as much width as nessary for their content

2.What is the margin, border, and padding?

Margin is the space outside the border of an element. It creates a gap between the element and other element around it. Border is the outline that wrap around the padding snf content of an element. It lies between the margin and the padding. Padding is the space between the content of an element and its borders. It creates an inners spac between the content and the element's border

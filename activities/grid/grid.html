What is the main different between CSS Grid and CSS Flex?

CSS Grid is the parent determines the size of the child. For anther example, when you need a two-dimensional layout where you want to control both rows and columns such as complex web page layout.

CSS Flex is the child want to become that size of the parent. For another example, when you need to a one-dimensinal layout such as aligning items in a row or a column.

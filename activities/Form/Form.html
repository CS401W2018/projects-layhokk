The form tag has an "action" attribute. What does it do?

The action tag specifies the URL where the form data should be sent when the form is submitted

The form tag has a "method" attribute. What does it do?

The method attribute defines the HTTP method used to send the form data. The two common values are GET and POST. Get is used to request data from a specifie resource. POST is generally used for submitting sensitive information, like passwords or personal data.

What is the difference between a "POST" and a "GET"?

GET appends data to the URL, making it visible in the address bar, while POST keeps data in the request body. For another example, GET is ideal for retrieving data without altering server state, while POST is used for actions like submitting forms or modifying data on the server.

Welcome

1. What is the difference between a block and an inline tag? Give examples of each one?

- Block tags can have their width and height adjusted and you can apply margins and padding to them. For example, div, p, ul,ol and li tag

- Inline tags cannot have their width and height set directly but can still apply margins and padding. For example, span, img, strong tags

2. What is normal flow in HTML?

- Normal flow provides the foundation of the web page layout before any advanced styling elements

3. What does float do? Why do you need to clear the float?

- The float is used to move an element to the left or right side. We need to clear the float to prevent layout issues that arise when floating elements. For example: clear: left; clear: right, clear: both

4. What do absolute, fixed, and relative do?

- Absolute used for creating precise layout, such as dropdown menus, and pop-ups

- Fixed is the element relative to the viewport, keeping it fixed even during scrolling and removes it from the normal flow.

- relative is positioned relative to its normal position in the flow. For example, position: relative; top: 10px, left: 20px

ASCII and Unicode

ASCII is an American standard code for information interchange. It is a character encoding using 7-bit binary numbers that represent text in computer.
Unicode is a universal character encoding that assigns number or character to support including alphabets, symbol and emojis from many language.

What is UTF-8?

UTF-8 represent every character in the Unicode. For example, English Letter, Number
